Pacific GuestHub - Sistema de Gestión de Reservas

Este repositorio contiene la planificación ágil y documentación técnica para el proyecto **Pacific GuestHub**, diseñado para automatizar los procesos de reserva del **Hotel Pacific Reef**.

## 📋 Información del Proyecto
* **Institución:** DUOC UC
* **Asignatura:** PRY3211 - Proyecto de Software
* **Estudiante:** Yolzer Awidan Saravia San Martín
* **Caso de Negocio:** Modernización de reserva bilingüe y pagos del 30%.

## 🛠️ Metodología
Se ha implementado el marco de trabajo **Scrum**, distribuyendo el desarrollo en **3 Sprints** de una semana cada uno para cumplir con el plazo de entrega de 21 días.

## 📂 Contenido del Repositorio
* **Planilla_SprintBacklog.xlsx**: Contiene el Product Backlog con 15 historias de usuario, estimación en puntos de historia (Fibonacci) y la descomposición del Sprint Backlog.
* **Project_Roadmap.xlsx**: Cronograma visual de las 3 semanas de ejecución.
* **Scrum_Project_Burndown.docx**: Informe técnico con la fundamentación (IEEE 830), análisis de roles y monitoreo del proyecto.
